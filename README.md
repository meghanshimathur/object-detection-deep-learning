# 🚀 Project 1: Faster R-CNN Object Detection

This project demonstrates a simplified implementation of the **Faster R-CNN architecture** for understanding the object detection workflow.

### 🔧 Key Components

- 🧠 CNN Backbone for feature extraction  
- 📦 Region Proposal Network (RPN)  
- 🔍 ROI feature extraction  
- 🎯 Detection head for classification  
- 📏 Bounding box regression  
- 🎥 Image and video prediction visualization  

### 📘 Concepts Covered

- Object detection pipelines  
- Feature map extraction  
- Region proposal networks  
- Bounding box prediction  
- Computer vision model workflows

# 🏥 Project 2: Breast Cancer Ultrasound Classification

This project focuses on **medical image classification** using **transfer learning with EfficientNetB0**.

The model classifies ultrasound images into three categories:

- 🟢 Benign  
- 🔴 Malignant  
- ⚪ Normal  

### 🔬 Key Features

- ⚡ Transfer learning using EfficientNetB0  
- 🔄 Data augmentation using ImageDataGenerator  
- ⚖ Handling class imbalance using class weights  
- 📊 Model evaluation using ROC curves and AUC score  
- 📉 Confusion matrix and classification report  
- 🔎 Explainable AI using Grad-CAM
